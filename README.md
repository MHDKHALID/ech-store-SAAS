```bash
# Python 2.x

# Open http://localhost:8000
```

**Using Node.js:**
```bash

# Run server
```

#### 3ï¸âƒ£ **Checkout**

- Enter card details

## ğŸ¨ Customization

### Color Scheme

Update CSS variables in `css/style.css`:

```css
:root {
    /* Primary Colors */
    --primary-color: #6366f1;      /* Main brand color */
    --primary-dark: #4f46e5;       /* Hover states */
    --primary-light: #818cf8;      /* Accents */
    
    /* Semantic Colors */
    --success: #10b981;            /* Success messages */
    --danger: #ef4444;             /* Errors, delete */
    --warning: #f59e0b;            /* Warnings */
    
    /* Gradients */
    --gradient-primary: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
    --gradient-secondary: linear-gradient(135deg, #f093fb 0%, #f5576c 100%);
}
```

### Typography

Change font family in `css/style.css`:

```css
body {
    font-family: 'Inter', -apple-system, BlinkMacSystemFont, 
                 'Segoe UI', Roboto, sans-serif;
}
```

To use a different Google Font:
1. Visit [Google Fonts](https://fonts.google.com/)
2. Select your font
3. Copy the `<link>` tag
4. Add to `<head>` section of HTML files
5. Update CSS `font-family`

### Layout & Spacing

Adjust grid columns in `css/style.css`:

```css
.products-grid {
    grid-template-columns: repeat(auto-fill, minmax(280px, 1fr));
    gap: 2rem; /* Adjust spacing */
}
```
